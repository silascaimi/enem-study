# Formularios rápidos: Matemática, Física e Química

Use este arquivo como revisão curta antes de listas, simulados e correções. Não decore tudo de uma vez: conecte cada fórmula a uma questão real.

## Matemática

| Tema | Fórmula/ideia | Atenção no ENEM |
|---|---|---|
| Porcentagem | p% = p/100 | Use fator multiplicativo: aumento de 20% = x 1,20; desconto de 20% = x 0,80. |
| Regra de três | a/b = c/d | Verifique se as grandezas são diretas ou inversas. |
| Juros simples | J = C.i.t; M = C + J | Confira se taxa e tempo estao na mesma unidade. |
| Juros compostos | M = C(1+i)^t | Muito usado em crescimento, desconto e comparação de propostas. |
| Média | soma dos valores / quantidade | Cuidado com média ponderada. |
| Probabilidade | P = casos favoráveis / casos possíveis | Conte possibilidades sem repetir. |
| Área do retangulo | A = b.h | Padronize unidades antes de calcular. |
| Área do triangulo | A = b.h/2 | Identifique base e altura reais. |
| Circunferencia | C = 2.pi.r | Diferencie raio, diametro e comprimento. |
| Volume do prisma | V = área da base . altura | Volume exige unidade cubica. |

## Física

| Tema | Fórmula/ideia | Atenção no ENEM |
|---|---|---|
| Velocidade média | v = d/t | Converter km/h e m/s quando necessário. |
| Movimento uniforme | s = s0 + v.t | Interpretar gráfico antes da conta. |
| Aceleração | a = delta v / delta t | Observar variação de velocidade. |
| Força resultante | F = m.a | Unidades: kg, m/s2 e newton. |
| Peso | P = m.g | Diferencie massa e peso. |
| Trabalho | W = F.d.cos(theta) | No ENEM, muitas vezes aparece como energia transferida. |
| Energia potencial | E = m.g.h | Associar a altura e conservação de energia. |
| Potência | P = E/t | Conta de luz usa potência e tempo. |
| Lei de Ohm | U = R.i | Circuitos simples, chuveiro e lampadas. |
| Potência elétrica | P = U.i | Também pode aparecer como P = R.i2 ou P = U2/R. |
| Calor sensivel | Q = m.c.deltaT | Cuidado com Celsius e variação de temperatura. |

## Química

| Tema | Fórmula/ideia | Atenção no ENEM |
|---|---|---|
| Mol | n = m/M | Massa molar deve estar em g/mol. |
| Concentração comum | C = m/V | Conferir se volume está em litro. |
| Diluição | C1.V1 = C2.V2 | Quantidade de soluto permanece constante. |
| Densidade | d = m/V | Pode aparecer em separação de misturas. |
| pH | pH = -log[H+] | pH menor indica maior acidez. |
| Balanceamento | Conservação dos atomos | Balancear antes da estequiometria. |
| Estequiometria | Proporção entre coeficientes | Converter massa para mol antes de comparar. |
| Termoquímica | Delta H < 0 exotérmica; Delta H > 0 endotérmica | Relacionar energia, combustíveis e ambiente. |
| Eletroquímica | Oxidação perde elétrons; redução ganha elétrons | Pilhas e baterias aparecem em contexto tecnológico. |
