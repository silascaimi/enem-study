# Formularios rapidos: Matematica, Fisica e Quimica

Use este arquivo como revisao curta antes de listas, simulados e correcoes. Nao decore tudo de uma vez: conecte cada formula a uma questao real.

## Matematica

| Tema | Formula/ideia | Atencao no ENEM |
|---|---|---|
| Porcentagem | p% = p/100 | Use fator multiplicativo: aumento de 20% = x 1,20; desconto de 20% = x 0,80. |
| Regra de tres | a/b = c/d | Verifique se as grandezas sao diretas ou inversas. |
| Juros simples | J = C.i.t; M = C + J | Confira se taxa e tempo estao na mesma unidade. |
| Juros compostos | M = C(1+i)^t | Muito usado em crescimento, desconto e comparacao de propostas. |
| Media | soma dos valores / quantidade | Cuidado com media ponderada. |
| Probabilidade | P = casos favoraveis / casos possiveis | Conte possibilidades sem repetir. |
| Area do retangulo | A = b.h | Padronize unidades antes de calcular. |
| Area do triangulo | A = b.h/2 | Identifique base e altura reais. |
| Circunferencia | C = 2.pi.r | Diferencie raio, diametro e comprimento. |
| Volume do prisma | V = area da base . altura | Volume exige unidade cubica. |

## Fisica

| Tema | Formula/ideia | Atencao no ENEM |
|---|---|---|
| Velocidade media | v = d/t | Converter km/h e m/s quando necessario. |
| Movimento uniforme | s = s0 + v.t | Interpretar grafico antes da conta. |
| Aceleracao | a = delta v / delta t | Observar variacao de velocidade. |
| Forca resultante | F = m.a | Unidades: kg, m/s2 e newton. |
| Peso | P = m.g | Diferencie massa e peso. |
| Trabalho | W = F.d.cos(theta) | No ENEM, muitas vezes aparece como energia transferida. |
| Energia potencial | E = m.g.h | Associar a altura e conservacao de energia. |
| Potencia | P = E/t | Conta de luz usa potencia e tempo. |
| Lei de Ohm | U = R.i | Circuitos simples, chuveiro e lampadas. |
| Potencia eletrica | P = U.i | Tambem pode aparecer como P = R.i2 ou P = U2/R. |
| Calor sensivel | Q = m.c.deltaT | Cuidado com Celsius e variacao de temperatura. |

## Quimica

| Tema | Formula/ideia | Atencao no ENEM |
|---|---|---|
| Mol | n = m/M | Massa molar deve estar em g/mol. |
| Concentracao comum | C = m/V | Conferir se volume esta em litro. |
| Diluicao | C1.V1 = C2.V2 | Quantidade de soluto permanece constante. |
| Densidade | d = m/V | Pode aparecer em separacao de misturas. |
| pH | pH = -log[H+] | pH menor indica maior acidez. |
| Balanceamento | Conservacao dos atomos | Balancear antes da estequiometria. |
| Estequiometria | Proporcao entre coeficientes | Converter massa para mol antes de comparar. |
| Termoquimica | Delta H < 0 exotermica; Delta H > 0 endotermica | Relacionar energia, combustiveis e ambiente. |
| Eletroquimica | Oxidacao perde eletrons; reducao ganha eletrons | Pilhas e baterias aparecem em contexto tecnologico. |

## Como revisar em 15 minutos

1. Escolha 5 formulas que apareceram nos seus erros.
2. Escreva uma frase explicando quando usar cada uma.
3. Resolva 3 questoes curtas.
4. Atualize o caderno de erros se repetir o mesmo problema.
